# Commit Message Skill Evaluation

Promptfoo-based evaluation for the commit-message skill using LLM-as-judge to assess commit message quality.

## Setup

1. **Install dependencies** (already done):
   ```bash
   npm install
   ```

2. **Set API key** according to the providers defined in the promptfooconfig.yaml

## Running Evals

```bash
# Run evaluation
npx promptfoo eval

# View results in UI
npx promptfoo view
```
